## 描述

这是个关于OS总结的工程，Q1代表最低端的8051内核、stm8内核的实时系统OS

## 组成
- M1: 8051 RTX-51系统
- M2: STM8 uCOSII系统
- M3: 

## 说明

OS-Q1项目针对低端的处理器，为实现多任务的调度和和响应应用层代码统一，至于是否需要在低性能平台上使用OS就见仁见智了。
